Ruutino Privacy Policy

Last updated: December 2025

This Privacy Policy explains how Ruutino (the "App") collects, uses, stores, and protects user data. By using the App, you agree to the practices outlined below.

1. Data We Collect

Account Information

  • Email address and password when users create an account.
  • Family or group identifiers if using shared features.

App Usage Data

  • Task creation, completion history, points, and leaderboard data.
  • Avatar customization and progression.

Device & Technical Data

  • Device type, OS version, crash logs, and analytics.

Optional Purchases

  • RevenueCat handles subscriptions; no payment information is stored by Ruutino.

Firebase Data

  • Authentication information.
  • Database entries for tasks, rewards, and user data.

2. How We Use Data

  • Account creation and authentication.
  • Syncing tasks, chores, points, and avatar settings.
  • Managing family groups and leaderboards.
  • Processing subscription features.
  • Improving app performance and reliability.

3. How We Store & Protect Data

  • Firebase security rules enforce protection of stored data.
  • Passwords are securely hashed.
  • All data is encrypted in transit (HTTPS) and at rest.

4. Sharing of Data

We do not sell or share personal data with third parties.

Data is only shared with:

  • Firebase for authentication and database hosting.
  • RevenueCat for subscription validation.
